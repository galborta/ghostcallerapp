Meditation Music App - Application Flow Document

Overview

This document maps out the complete user flow through the Meditation Music App, describing each screen, transition, and user action in specific detail.

App Entry & Authentication

App Launch Screen

When a user opens the app, they briefly see the launch screen displaying the app logo and name. This screen automatically transitions to either the Login Screen or Home Screen depending on authentication status.

Login Screen

The Login Screen appears for users who are not logged in. It displays the app logo at the top, followed by three login options: Email/Password, Google Sign-In, and Apple Sign-In. At the bottom is a "Create Account" link. If the user arrived via an artist referral link, a banner displays "30% discount applied to your first month when you create an account."

Sign-Up Screen

Users who tap "Create Account" see the Sign-Up Screen with fields for email, password, and password confirmation. A checkbox for agreeing to terms and privacy policy appears at the bottom. The discount banner remains visible if the user came from a referral link. After submitting valid information, users move to the Onboarding flow.

Password Reset Screen

Users who tap "Forgot Password" on the Login Screen arrive at this page. It contains an email input field and a "Reset Password" button. After submission, a confirmation message appears instructing the user to check their email.

Onboarding Flow

Welcome Screen

First-time users see a welcome message with the app name and tagline. A "Get Started" button at the bottom takes them to the next onboarding screen.

App Introduction Screen

This screen consists of 3-4 swipeable cards that explain core features:

  1. Artist-centered meditation music
  2. Guided and music-only options
  3. GitHub-style calendar tracking
  4. Sharing capabilities

Each card has an illustrative image, heading, and brief description. Page indicators appear at the bottom, along with "Next" and "Skip" buttons.

Referral Confirmation (Conditional)

Users who came through a referral link see a confirmation screen showing: "30% discount applied to your first month! Your account is linked to [Artist Name]." A smiling artist image appears with a success message. A "Continue" button leads to the Artist Selection screen.

Main App Navigation

Home Screen (Artist Selection)

The Home Screen is the primary landing screen after login. It displays:

  • Search bar at the top for finding artists
  • Horizontally scrollable row of recently played artists (if any)
  • Grid of featured artists with their photos
  • Bottom navigation bar with Home, Calendar, and Settings tabs

Each artist card shows the artist's name and image. Tapping an artist card navigates to the Artist Detail Screen.

Artist Detail Screen

This screen shows comprehensive information about a selected artist:

  • Large artist image at the top
  • Artist name and short bio
  • "Start Meditation" button
  • Full biography in a scrollable section
  • List of available meditation tracks with duration indicators
  • Visual indicators showing which tracks are guided vs. music-only

Tapping "Start Meditation" leads to the Session Setup Screen. Tapping a specific track also leads to Session Setup with that track pre-selected.

Session Setup Screen

This modal screen appears over the Artist Detail Screen and shows:

  • Selected artist name and image
  • Duration options (based on available track lengths)
  • Toggle switch between guided and music-only options (all tracks can be experienced either way based on user preference)
  • Preview button to hear a sample
  • "Begin Meditation" button at the bottom

After making selections and tapping "Begin Meditation," the user transitions to the Meditation Player Screen.

Meditation Player Screen

This full-screen experience shows:

  • Large countdown timer in the center showing remaining time
  • Artist name and track title
  • Background image (subtle, non-distracting)
  • Play/pause button
  • Volume control
  • "End Session" button

When the session ends naturally or is manually ended, the user sees the Session Completion Screen.

Session Completion Screen

This celebratory screen shows:

  • "Session Complete" message
  • Session statistics (duration meditated)
  • Updated streak information
  • Confetti animation (subtle)
  • "Share This Session" button
  • "Return Home" button
  • "Start Another Session" button

Tapping "Share This Session" opens the Session Sharing modal. "Return Home" takes the user back to the Home Screen. "Start Another Session" returns to the Session Setup Screen.

Calendar & Tracking

Calendar Screen

Accessed via the Calendar tab in the bottom navigation, this screen shows:

  • GitHub-style calendar visualization with color-coded cells
  • Toggle between monthly and yearly views
  • Color intensity legend explaining the shading system
  • Current streak counter
  • Longest streak record
  • Total minutes meditated
  • "Share Calendar" button at the top right

Tapping any colored cell opens the Day Detail view. Tapping "Share Calendar" opens the Calendar Sharing modal.

Day Detail View

This modal appears when a user taps a specific day on the calendar and shows:

  • Selected date
  • List of all meditation sessions completed that day
  • Each session entry shows: artist name, track name, duration, and timestamp
  • Total minutes meditated that day
  • Close button to return to the Calendar Screen

Sharing Features

Session Sharing Modal

This modal appears when a user chooses to share a completed session and includes:

  • Pre-designed card showing artist information summary and how many minutes the meditation session lasted
  • Optional text field for adding a personal message
  • Platform selection buttons (Instagram, Twitter, Facebook, etc.)
  • "Save to Photos" option
  • "Cancel" button

After selecting a sharing method, the appropriate platform app opens with the card pre-loaded.

Calendar Sharing Modal

This modal appears when a user chooses to share their calendar and includes:

  • Preview of the calendar visualization
  • Selection options for sharing a specific day, month, or year view
  • Statistics summary (streak, total minutes)
  • Optional text field for adding a personal message
  • Platform selection buttons (Instagram, Twitter, Facebook, etc.)
  • "Save to Photos" option
  • "Cancel" button

After selecting a sharing method, the appropriate app opens with the calendar image pre-loaded.

Settings & Profile

Settings Screen

Accessed via the Settings tab in the bottom navigation, this screen shows:

  • User profile information (name, email, avatar)
  • "Edit Profile" button
  • Referring artist information (if applicable)
  • "Create Referral Link" button
  • Current referral discount status (X% lifetime discount)
  • Number of successful referrals (X/5 for Superfan status)
  • Notification preferences toggle
  • App version information
  • "Log Out" button
  • "Terms of Service" and "Privacy Policy" links

Tapping "Edit Profile" opens the Profile Edit Screen. Tapping "Create Referral Link" generates a shareable link. Tapping "Log Out" shows a confirmation dialog before logging the user out.

Profile Edit Screen

This screen allows users to modify basic profile information:

  • Profile picture (with option to change)
  • Name field
  • Email field (non-editable if using social login)
  • "Save Changes" button
  • "Cancel" button

After saving changes, the user returns to the Settings Screen with updated information.

Error & Status Screens

Network Error Screen

This screen appears when the app cannot connect to the internet and shows:

  • Error illustration
  • "Connection Error" heading
  • Brief explanation text
  • "Try Again" button

Tapping "Try Again" attempts to reload the previous screen.

Content Loading Screen

This screen appears briefly while content is loading and shows:

  • Loading spinner or pulsing animation
  • "Loading..." text
  • Progress indicator for larger content loads

Once loading completes, the user automatically transitions to the requested content screen.

Transition Flows

First-Time User Flow

App Launch → Login Screen → Sign-Up Screen → Welcome Screen → App Introduction → Referral Confirmation (if applicable) → Home Screen

Returning User Flow

App Launch → Login Screen → Home Screen

Meditation Session Flow

Home Screen → Artist Detail Screen → Session Setup Screen → Meditation Player Screen → Session Completion Screen → (Share or Return Home)

Calendar Sharing Flow

Home Screen → Calendar Tab → Calendar Screen → "Share Calendar" button → Calendar Sharing Modal → External sharing app

Logout Flow

Any Screen → Settings Tab → Settings Screen → "Log Out" button → Confirmation Dialog → Login Screen

Edge Cases & Special States

Session Interruption

If a meditation session is interrupted by a call or other system event, the session pauses automatically. When the user returns to the app, they see a "Resume Session" dialog with options to continue or end the session.

Artist Referral Attribution

When a user clicks an artist's referral link and installs the app, their account is permanently linked to that artist in the database. This attribution persists even if the user doesn't complete signup immediately, as long as they use the same device.

User Referral Program

When a user creates a referral link and shares it, the system tracks all sign-ups through that link. For each new paying member that subscribes through their link, the referring user receives a 10% lifetime discount on their subscription, up to a maximum of 50% (after 5 successful referrals). Once a user reaches 5 or more successful referrals, they achieve "Superfan" status, which is displayed on their profile.

Failed Login

If login fails due to incorrect credentials, the Login Screen displays an error message below the relevant field. The login button remains available for retry.

Background Audio

When the user navigates away from the Meditation Player Screen or locks their device, audio continues playing. A notification appears in the system tray showing the session in progress with basic controls (pause/resume).

Calendar Date Selection

When viewing the calendar, the current date is highlighted with a distinct border. Days with no meditation activity appear as empty cells. Future dates appear slightly dimmed to indicate they cannot yet contain meditation data.
